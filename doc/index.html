String

Here are some helper functions for strings.

isEmpty

Check if a given string is empty.

isEmpty(string: string): boolean
Parameters
string (string) The string to check.
Returns
boolean: Will evaluate to true if value is: null, undefined, NaN, empty string (""), 0 or false.

titleCase

Convert a given string to title case.

titleCase
Parameters
string (string) The string to convert to title case.
Returns
string: Returns the title case string.

properCase

Convert a string to the proper case.

properCase(string: string): string
Parameters
string (string) The string to convert to title case.
Returns
string: Returns the title case string.

Date

Here are some helper functions for dates.

isFuture

Check if a given date is in the future.

isFuture
Parameters
date (timestamp) The date to check.
Returns
boolean: The date is in the future.

isAfter

Check if the first date is after the second one.

isAfter
Parameters
date (timestamp) The date that should be after the other one to return true.
dateToCompare (timestamp) The date to compare with.
Returns
boolean: The first date is after the second date.

Misc

Here are some helper functions for dates.

alert

Display an alert message.

alert(title: string, text: string, type: string): html
Parameters
title (string) The title of the modal.
text (string) A description for the modal.
type (string) The type of the modal. You can choose between five built-in types which will show a corresponding icon animation: warning, error, success, info, question.
Returns
html: Display the alert message.

inArray

Checks if predicate returns truthy for any element of collection. Iteration is stopped once predicate returns truthy.

inArray(collection: Array, predicate: Function): boolean
Parameters
collection (Array) The collection to iterate over.
predicate (Function) The function invoked per iteration.
Returns
boolean: Returns true if any element passes the predicate check, else false.
